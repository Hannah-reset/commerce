# commerce
项目描述：
利用java语言，MySQL，SpringBoot框架，HTML，CSS,JavaScript，jQuery，mybatis等技术实现一个包含前端与后端的模拟电商项目。
项目设计借鉴了b站关于电商项目的相关视频。主要功能与淘宝相同，相关数据自拟。
<properties>
	<java.version>1.8</java.version>
</properties>
